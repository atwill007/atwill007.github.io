安装Xcode

  1. 安装 Xcode 9.0 以上版本
  2. 配置Xcode命令行工具以使用最新Xcode版本 sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer
  3. 确保已经同意Xcode许可协议或通过命令行 sudo xcodebuild -license 同意

    设置iOS模拟器

    lib | 是flutter的dart方面的代码(该目录下有个项目入口文件main.dart,可直接运行)
    build | 项目的构建输出目录
